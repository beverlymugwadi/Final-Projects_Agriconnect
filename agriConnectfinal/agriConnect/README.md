# AgriConnect

## Prerequisites

Ensure you have the following installed on your system:

- Node.js (for frontend and backend)
- MongoDB (if a database is required)
- Docker (if the project is containerized)

## Dependencies

### Backend Dependencies

```
axios
bcryptjs
cors
dotenv
express
express-async-handler
jsonwebtoken
mongoose
morgan
multersocket.io
```

### Frontend Dependencies
```
@ant-design/icons
antd
axios
cra-template
date-fns
react
react-dom
react-router-dom
react-scripts
socket.io-client
web-vitals
```

## Installation

1. Backend Setup:
   
   cd backend
   npm install
   

2. Frontend Setup:
  
   cd frontend
   npm install
 

## Running the Project

1. Start the backend server:

   cd backend
   npm start


2. Start the frontend development server:

   cd frontend
   cd my-react-app
   npm start
